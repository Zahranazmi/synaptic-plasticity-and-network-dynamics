# Synaptic Plasticity and Network Dynamics

This repository explores simplified computational models of synaptic transmission, feedforward neural dynamics, and activity-dependent plasticity mechanisms.

The notebooks progress from spike propagation in small feedforward networks toward spike-timing dependent plasticity (STDP) and timing-sensitive learning dynamics.

The focus is on mechanistic intuition and computational modeling rather than highly detailed biophysical realism.

---

## Development Context

These notebooks were developed progressively during self-study and thesis preparation in computational neuroscience. The repository was later curated and reorganized into a structured public portfolio focused on learning dynamics and mechanistic neural modeling.

---

## What This Repository Demonstrates

This repository demonstrates:
- feedforward neural simulations,
- spike-based synaptic interactions,
- temporal summation,
- short-term synaptic dynamics,
- spike-timing dependent plasticity (STDP),
- timing-sensitive weight updates,
- parameter exploration,
- and mechanistic interpretation of simplified learning models.

---

## Repository Structure

### 01 — Feedforward Neural Dynamics
Introduces spike propagation and synaptic transmission between connected neurons.

### 02 — Short-Term Synaptic Dynamics
Explores activity-dependent short-term synaptic changes and temporal effects.

### 03 — Spike-Timing Dependent Plasticity
Implements STDP-based learning rules and timing-sensitive synaptic updates.

### 04 — Timing-Dependent Learning Dynamics
Examines how learning behavior changes as a function of spike timing and model parameters.

---

## Scientific Scope and Limitations

The models in this repository are simplified phenomenological models intended for conceptual and educational purposes.

They should not be interpreted as biologically complete simulations of synaptic physiology or hippocampal circuitry.
